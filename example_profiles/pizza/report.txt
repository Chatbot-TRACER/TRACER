## FUNCTIONALITIES (Workflow Structure)
- provide_welcome_message: Welcomes the user to Fast Pizza and states its ability to help order pizzas and drinks. | Category: Chatbot Meta | Outputs: [welcome_message: Welcoming message and statement of capabilities.]
- provide_information_capabilities: Informs the user about the types of questions it can answer. | Category: Chatbot Meta | Outputs: [information_capabilities: List of topics about which the chatbot can provide information, including opening hours, types of pizza, prices, order pickup, and estimated time for orders.]
- provide_opening_hours: Provides the opening hours of the establishment. | Category: Information Providing | Outputs: [opening_hours: The daily opening and closing times.]
- list_pizza_options: Lists the available pizza options and types, including custom options. | Category: Information Providing | Outputs: [pizza_types: A list of available pizza types, including standard options like Margarita, Carbonara, Marinera, Hawaiian, Four cheese, Vegetarian, and the option to create a custom pizza.]
- confirm_order_placement_and_prompt_for_pizza_type: Confirms the user can place an order and prompts them to choose a pizza type from the menu. | Category: Order Placement | Outputs: [pizza_types: margarita, carbonara, marinera, hawaiian, four cheese, and vegetarian]
  - prompt_for_pizza_size_and_type: Prompts the user to specify the desired size and type of pizza they want to order, providing options for both. | Category: Order Placement | Params: [pizza_size: The desired size of the pizza. [options: large, medium, small]] | Outputs: [pizza_options: margarita, carbonara, marinera, hawaiian, four cheese, and vegetarian]
    - confirm_pizza_order: Confirms the pizza order based on size and type provided by the user. | Category: Order Placement
      - prompt_for_drink_order: Prompts the user to specify the number and type of drinks they would like to order, providing options for drink types and confirming order details as needed. | Category: Order Placement | Params: [number_of_drinks: The number of drinks the user wants to order.; drink_type: The type of drink the user wants (e.g., coke, sprite, water). [options: coke, sprite, water]]
        - provide_order_summary_and_total_cost: Provides a summary of the order and states the total cost. | Category: Order Confirmation | Outputs: [order_summary: Summary of the items included in the order.; total_cost: The total cost of the order.]
- inform_contact_information_unavailable: Informs the user that the requested contact information is unavailable and suggests alternative sources to find it, such as the restaurant's website or online search. | Category: Information Providing
- inform_delivery_options_unavailable: Informs the user that delivery options for the restaurant are unavailable and suggests checking their website or contacting them directly. | Category: Information Providing
- inform_pizza_pricing_unavailable: Informs the user that pricing information for pizzas is unavailable and suggests contacting the restaurant directly. | Category: Information Providing | Outputs: [price_unavailable: Indicates that the chatbot is unable to provide pricing details and advises contacting the restaurant for pricing information.]
