=== CHATBOT FUNCTIONALITY ANALYSIS ===

## FUNCTIONALITIES (Workflow Structure)
- provide_menu_options: The chatbot provides an overview of available pizza and drink options, including predefined and custom pizzas.
  - request_predefined_menu: User requests the predefined pizza menu to select a pizza type.
    - list_pizza_options: User inquires about the types of pizzas available, chatbot provides the list of predefined pizza options.
      - select_pizza_size: User is prompted to select the size of the pizza for their order. (Params: pizza_size)
        - confirm_order: Chatbot confirms the user's order details including the pizza size, type, and additional drinks. (Params: pizza_size, pizza_type, drink_quantity, drink_type)
          - add_drinks_to_order: User specifies the quantity and types of drinks to include in their order. (Params: drink_quantity, drink_type)
            - provide_order_summary: Chatbot provides a summary of the user's order, including total cost, preparation time, and order ID. (Params: pizza_type, drink_quantity, total_cost, preparation_time, order_id)
            - confirm_drink_selection: Confirm the user's selection of drinks for the order. (Params: drink_name)
        - query_order_cost: User inquires about the cost of a specific pizza size and type. (Params: pizza_type, pizza_size)
        - calculate_order_cost: User requests the total cost of a specific pizza order, in this case, a medium Hawaiian pizza, and the chatbot provides the necessary pricing based on size and type. (Params: pizza_type, size)
      - request_hawaiian_pizza_ingredients: The user repeatedly asks for details on the ingredients in the Hawaiian pizza, seeking to understand its components.
        - handle_repeat_requests: The chatbot fails to provide the requested information about the Hawaiian pizza ingredients, asking the user to rephrase their question multiple times.
    - list_custom_pizza_toppings: User requests information about the available toppings for a custom pizza; chatbot provides the available options.
  - list_drink_options: User inquires about the types of drinks available, and chatbot provides the list of options and prices.
    - provide_drink_prices: Chatbot provides information on the prices of available beverages.
- provide_information_abount_ordering: Answer questions related to ordering procedures, pizza varieties, prices, and order delivery time.
- provide_opening_hours: The chatbot provides information about the establishment's operating hours.
- list_beverages: The chatbot provides information on available beverages and their prices.
- provide_pizza_prices: The chatbot details the prices of different types of pizzas, including custom pizzas and additional ingredient costs.
- provide_order_time: The chatbot estimates the typical time it takes to get an order based on the number of pizzas and shop busyness.
- explain_order_pickup: The chatbot explains the process of picking up an order and the shop's location.
