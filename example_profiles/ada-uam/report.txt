=== CHATBOT FUNCTIONALITY ANALYSIS ===

## FUNCTIONALITIES (Workflow Structure)
- list_services: The chatbot lists the main services provided by the UAM's Information Technologies.
- provide_faculty_info: The chatbot provides information about the faculties and schools at UAM and available academic programs.
- select_service_of_interest: The user selects the button to learn more about UAM services of interest.
- select_future_students: The user selects the button to obtain information intended for future students.
- provide_software_info: The chatbot provides a link to information about software available for students.
- provide_contact_info: The chatbot provides information on how to contact the Centro de Atención a Usuarios (CAU) through email, phone, and in person, including the operating hours.
- provide_student_software_info: The chatbot provides a link to access information about the software available for students at UAM.
- view_university_information: The chatbot provides information about the Universidad Autónoma de Madrid and links to faculties and programs.
- select_vida_universitaria: The user selects the button to view information about university life.
- select_sede_electronica: The user selects the button to access information regarding the electronic headquarters for academic certificate requests.
- request_academic_certificate_info: The chatbot provides details on how to request the Academic Certificate, including access methods for current and former students.
  - repeat_request_academic_certificate_info: The user repeatedly selects the option to receive information on how to request the Academic Certificate.
- select_academic_certificate_request: The user selects the button to initiate the request for an Academic Certificate.
- provide_resources_for_international_students: The chatbot asks the user to rephrase their question or directs them to a catalog of services related to resources for international students at UAM.
- provide_technology_services_info: Offers information about the technological services provided by the UAM and solicits further specific queries from the user.
- request_clarification: Asks the user to rephrase their question or directs them to consult the catalog of services when responses are repetitive or unclear.
- explain_user_support_services: The chatbot explains the types of help or services that can be obtained from the Centro de Atención a Usuarios (CAU).
- provide_technical_support_info: The chatbot informs the user to open a ticket in the CAU for technical support services by detailing their request via email.
- repeat_admission_info: The chatbot reiterates that it does not have information about degree admissions and provides the same contact information as before.
- encourage_ticket_submission: Advises the user to open a support ticket with the CAU for specific inquiries, prompting them to explain their request in detail via email.
