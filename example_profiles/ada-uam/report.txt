## FUNCTIONALITIES (By Category)

### CATEGORY: Account & Access (2 functions)
- provide_uam_identity_management_policy_information: Provides information about the UAM's identity management and logical access control policy, including email and service usage extensions after the end of the binding relationship with the university. It also provides links to FAQs for saving email content and contact information for the CAU.
- provide_id_uam_account_recovery_information: Provides information on how to recover a forgotten ID-UAM user ID or password, including links to search for the ID by DNI, contact the CAU, generate an activation code, or change the password with an activation code. Also provides a link to detailed instructions.

### CATEGORY: Chatbot Meta (1 functions)
- request_rephrasing_or_present_service_catalog: Asks the user to rephrase their question or offers to direct them to the service catalog.

### CATEGORY: General Information (5 functions)
- inform_holiday_hours_unavailable: Informs the user that information about holiday hours is not available from the IT service and suggests consulting the relevant school or faculty.
- provide_uam_description_and_faculty_list_link: Provides a general description of the Universidad Autónoma de Madrid and a link to a list of faculties and schools.
- provide_directory_update_information: Provides information about UAM directory inclusion criteria and contact information for PTGAS, PDI, and Personal Investigador services to facilitate directory information updates.
- present_directory_options: Presents options related to the UAM contact directory (e.g., updating data, finding emails) and allows the user to select an action.
- provide_uam_contact_directory_link: Provides a link to the UAM contact directory (PÁGINAS BLANCAS) where users can find contact information for UAM personnel.

### CATEGORY: Helpdesk Information (1 functions)
- provide_cau_contact_information: Provides contact information for the Centro de Atenci\u00f3n a Usuarios (CAU), including email, phone number, physical address, and operating hours, including standard, holiday, and suspension hours.

### CATEGORY: Software & Applications (2 functions)
- suggest_teams_help_resources: Suggests consulting TEAMS tutorials and provides button links related to common TEAMS issues and help options.
- inform_about_teams_telephony_migration_and_redirect: Informs the user about the ongoing migration of telephony to Microsoft Teams and redirects them to a service request form or provides instructions on how to submit one.

### CATEGORY: Student Services (1 functions)
- provide_academic_certificate_information: Provides information on how to request an Academic Certificate from the Electronic Headquarters, differentiating between current students (access with UAM username and password), former students (access with Cl@ve), and former foreign students (contact the Secretariat of their Faculty or School).
