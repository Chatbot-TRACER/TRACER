=== CHATBOT FUNCTIONALITY ANALYSIS ===

## FUNCTIONALITIES (Workflow Structure)
- provide_services_overview: The chatbot provides an overview of the services offered by Tecnologías de la Información at UAM, referring the user to the official website for detailed information.
- provide_contact_info: The chatbot provides contact information for the Centro de Atención a Usuarios (CAU), including email, phone number, and physical address for assistance.
- provide_support_services: The chatbot explains the types of support services available for technical problems at UAM, specifically regarding videoconferencing connections and software.
- refer_to_service_catalog: The chatbot redirects the user to consult the service catalog for specific information on services offered by Tecnologías de la Información at UAM.
- address_specific_support_requests: The chatbot attempts to address user requests for specific support services but primarily refers back to contact information and the service catalog.
- provide_moodle_support_options: The chatbot offers options for specific issues related to Moodle, allowing the user to choose a particular problem they are experiencing.
- provide_password_change_instructions: The chatbot provides instructions on how to change the password for the ID-UAM account and how to obtain a code for activation.
- verify_user_identity: The chatbot offers information on how to retrieve the user identifier if forgotten and provides further instructions for changing the password.
- reiterate_password_change_process: The chatbot repeats the instructions for changing the password and emphasizes the importance of password management.
- clarify_support_for_moodle: The user seeks clarification on the type of technical help available for issues with Moodle, prompting a detailed referral to specific resources.
- request_alternate_expression: Requests the user to rephrase their inquiry if not clearly understood, while offering a link to the service catalog as a resource.
- offer_contact_support: Chatbot provides contact information for user support in case of further inquiries or issues.
- select_change_password: The user chooses the option to change their password, resulting in detailed instructions for retrieving their user ID and changing their password.
