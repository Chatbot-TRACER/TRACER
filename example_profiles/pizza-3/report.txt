=== CHATBOT FUNCTIONALITY ANALYSIS ===

## FUNCTIONALITIES (Workflow Structure)
- inquire_main_functionality: User asks what main functionalities the chatbot offers.
  - provide_predefined_pizza_options: Chatbot lists available predefined pizzas for the user to choose from.
    - select_pizza_size: User specifies the size of the pizza they wish to order. (Params: pizza_size)
      - confirm_pizza_order: Chatbot acknowledges the user's pizza choice and requests drink order information.
        - confirm_order_details: Chatbot summarizes the order details, including total cost, pickup time, and location.
          - add_item_to_cart: The user expresses interest in adding another item to their order, such as drinks. (Params: item_name)
          - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
        - provide_drink_options: Chatbot lists available drink options and their prices.
          - add_item_to_order: The chatbot enables the user to add a specific drink item to their pizza order. (Params: drink)
            - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
      - confirm_order: The user confirms their pizza order, including size and toppings, before finalizing. (Params: pizza_size, topping1, topping2, topping3)
  - inquire_custom_pizza_ingredients: Chatbot requests the user to specify size and toppings for custom pizza orders. (Params: pizza_size, toppings)
    - select_pizza_size: User specifies the size of the pizza they wish to order. (Params: pizza_size)
      - confirm_pizza_order: Chatbot acknowledges the user's pizza choice and requests drink order information.
        - confirm_order_details: Chatbot summarizes the order details, including total cost, pickup time, and location.
          - add_item_to_cart: The user expresses interest in adding another item to their order, such as drinks. (Params: item_name)
          - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
        - provide_drink_options: Chatbot lists available drink options and their prices.
          - add_item_to_order: The chatbot enables the user to add a specific drink item to their pizza order. (Params: drink)
            - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
      - confirm_order: The user confirms their pizza order, including size and toppings, before finalizing. (Params: pizza_size, topping1, topping2, topping3)
    - select_toppings: The user selects toppings for their pizza, choosing from a predefined list of available toppings. (Params: topping1, topping2, topping3)
      - confirm_pizza_order: Chatbot acknowledges the user's pizza choice and requests drink order information.
        - confirm_order_details: Chatbot summarizes the order details, including total cost, pickup time, and location.
          - add_item_to_cart: The user expresses interest in adding another item to their order, such as drinks. (Params: item_name)
          - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
        - provide_drink_options: Chatbot lists available drink options and their prices.
          - add_item_to_order: The chatbot enables the user to add a specific drink item to their pizza order. (Params: drink)
            - summarize_order: The chatbot summarizes the user's current order, including pizza and drink selections.
      - confirm_order: The user confirms their pizza order, including size and toppings, before finalizing. (Params: pizza_size, topping1, topping2, topping3)
- inquire_pizza_sizes: User asks about the sizes available for pizzas.
- inquire_delivery_status: User asks about the delivery options for their order.
