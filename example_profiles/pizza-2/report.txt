=== CHATBOT FUNCTIONALITY ANALYSIS ===

## FUNCTIONALITIES (Workflow Structure)
- provide_options: Chatbot presents a list of predefined pizza options for the user to choose from.
  - select_option: User chooses a specific pizza option from the list provided by the chatbot. (Params: pizza_type)
    - gather_information: Chatbot asks the user for the size of the pizza they would like to order.
      - process_request: Chatbot confirms the user’s pizza order based on the selected type and size. (Params: pizza_type, pizza_size)
        - gather_drinks_info: Chatbot requests the user to indicate how many drinks they would like to order.
    - process_request: Chatbot confirms the user’s pizza order based on the selected type and size. (Params: pizza_type, pizza_size)
      - gather_drinks_info: Chatbot requests the user to indicate how many drinks they would like to order.
